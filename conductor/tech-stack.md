# Technology Stack

## Core Technologies
- **TypeScript**: Used for both frontend and backend development to ensure type safety, improve maintainability, and provide a better developer experience.

## Frontend
- **Next.js (React)**: Employed as the primary frontend framework. Next.js offers features like server-side rendering (SSR), static site generation (SSG), and image optimization, which align well with the project's goal for a high-performance and SEO-friendly web application.

## Backend
- **Express.js (Node.js)**: Utilized as the backend framework to build a flexible and efficient API layer. Using Express with TypeScript allows for a unified language across the stack.

## Database
- **SQLite**: A lightweight, serverless, and file-based relational database. This choice is ideal for managing structured grant information while keeping the infrastructure simple and cost-effective, especially for the initial project phase.
