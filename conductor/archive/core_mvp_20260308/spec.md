# Specification: Core Grant Listing and User Dashboard MVP

## Goal
Build the core functionality for searching grants and a user dashboard to save them, supporting artists in finding funding opportunities.

## User Stories
- **Filter by Category**: As an artist, I want to filter grants by category (e.g., painting, sculpture) to find relevant opportunities.
- **Deadline Visibility**: As an artist, I want to see submission deadlines clearly displayed to manage my time.
- **Save Opportunities**: As an artist, I want to save interesting opportunities for later review.

## Key Features
- **Grant Search & Filter**: A searchable list of grants with filtering by category and deadline.
- **User Dashboard**: A personal space for users to view and manage their saved grants.

## Tech Stack
- **Frontend**: Next.js (React), TypeScript, CSS (Vanilla)
- **Backend**: Express.js (Node.js), TypeScript
- **Database**: SQLite

## Data Strategy
- **Initial Phase**: Manual entry or mock data based on information from X and Instagram.
